# 🧠 Project Synapse

**Your Second Brain - Never Lose an Idea Again**

An intelligent content management system that captures, understands, and makes searchable any content you save - from articles and products to handwritten notes and AI conversations.

## ✨ Features

- 🎯 **Smart Capture**: Save any content - URLs, screenshots, notes, products, videos
- 🤖 **AI-Powered Understanding**: Uses Claude API to analyze and extract metadata
- 🔍 **Semantic Search**: Natural language queries like "black shoes under $300"
- 🎨 **Beautiful Visual Cards**: Content-specific rendering (not just links!)
- ⚡ **Chrome Extension**: One-click save from any website
- 🧩 **Multi-Type Support**: Articles, products, videos, images, notes, code, and more

## 🛠 Tech Stack

- **Backend**: Node.js, Express.js, PostgreSQL with pgvector
- **AI**: Claude API (content analysis), Gemini (embeddings)
- **Frontend**: React, Tailwind CSS
- **Extension**: Chrome Extension (Manifest V3)
